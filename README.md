# Student Course Planner (Java)

## Overview
A Java application with a **console-based interface** that allows students to manage course schedules, detect time conflicts, and plan courses efficiently.

## Features
- Add and remove students and courses
- Detect time conflicts between courses
- View schedules
- Operates entirely via the console (command-line interface)

## Tech Stack
- Java 22
- Eclipse IDE

## How to Run
1. Clone the repository:
2. Open the project in your preferred IDE
3. Run `Main.java`
